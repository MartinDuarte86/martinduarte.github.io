# Enterprise Data Architecture Blueprint Service

Use this skill for `Enterprise Data Architecture Blueprint`.

- Reference architecture service.
- Goal: align business, information domains, technical principles, standards, and platform components.
- Preserve the pain points: dispersed decisions, inconsistent models, messy integrations, lack of common standards.
- Preserve deliverables: enterprise blueprint, high-level logical and physical architecture, principles and standards, integration guidelines, governance/security/quality recommendations, capability and domain map.
- Preserve benefits: alignment, less ad hoc decision-making, lower architectural debt, shared foundation for modernization and scale.
