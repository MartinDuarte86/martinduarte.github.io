# Data Platform Modernization Program Service

Use this skill for `Data Platform Modernization Program`.

- Broad transformation-program design service.
- Goal: move legacy or partially modern platforms toward cloud or lakehouse architectures with lower risk and better sequencing.
- Preserve the pain points: obsolete platforms, low scalability, high operational dependence, need for end-to-end modernization.
- Preserve deliverables: documented modernization program, detailed target architecture, phased migration plan, executive and technical roadmap, risk/dependency analysis, transition governance recommendations.
- Preserve benefits: planned transformation, stronger alignment, clarity on stages and scope, solid base for implementation or advisory.
