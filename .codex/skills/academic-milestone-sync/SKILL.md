# Academic Milestone Sync

Use this skill when editing the Academia section in the landing or related brochures.

## Rules

- Always list `Master en Inteligencia Artificial` from `Universidad de San Andres` first.
- Add `IA de Vanguardia` when the milestone is the Master.
- Add a visible `Universidad de Palermo` logo treatment for `Licenciado en Big Data`.
- Keep `IBM Architectural Thinking` inside `AWS / IBM Certification Stack`.
- Group it with:
  - `AWS Cloud Practitioner`
  - `AWS Data Analytics`
  - `Data Lakes on AWS`

## UI Guidance

- Use logo tiles, initials, or marks already compatible with the landing if official assets are unavailable.
- Preserve the hierarchy visually through card order, badge emphasis, and title weight.
