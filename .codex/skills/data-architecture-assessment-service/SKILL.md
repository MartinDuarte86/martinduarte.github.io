# Data Architecture Assessment Service

Use this skill for anything related to `Data Architecture Assessment`.

- Entry-point diagnostic service.
- Goal: understand the current architecture and reduce uncertainty before modernization or investment.
- Preserve the four-step framework: Business Data Context, Current Architecture Mapping (AS-IS), Framework Evaluation, Future Architecture & Roadmap (TO-BE).
- Preserve deliverables: executive assessment, AS-IS map, capability-gap diagnosis, dimension-based evaluation, high-level target architecture, prioritized roadmap.
- Preserve benefits: clarity, lower uncertainty, objective prioritization, foundation for Blueprint, Strategy, or Modernization.
